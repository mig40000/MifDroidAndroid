## üí° Key Concepts

### Smali
- Android bytecode assembly language
- Generated by APKTool from DEX files
- Human-readable format
- Analyzed to understand app behavior

‚Üí **Read:** Workflow-Explanation.md

### Backward Slicing
- Algorithm to trace register values
- Works backward through assembly code
- Finds where values are defined
- Reconstructs original content

‚Üí **Read:** Workflow-Explanation.md or Visual-Quick-Reference.md

### WebView Bridge
- addJavascriptInterface connects Java to JavaScript
- JavaScript can call exposed Java methods
- Security-critical area
- Needs analysis

‚Üí **Read:** Workflow-Explanation.md

### loadUrl
- WebView method to load JavaScript/URLs
- Analyzed to understand what's loaded
- Primary focus of this analyzer
- Security implications

‚Üí **Read:** Executive-Summary.md or Workflow-Explanation.md
